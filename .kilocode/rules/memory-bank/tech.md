# Tecnologias e Dependências

**Principais tecnologias:**
- Vue.js 3.5.13 (frontend SPA)
- Node.js (scripts utilitários e backend)
- Firebase 11.10.0 (autenticação, banco de dados, deploy, hosting)
- SCSS/Sass 1.76.0 (estilização modular)
- Pinia 2.3.0 (gerenciamento de estado)
- Vite 5.4.19 (build e desenvolvimento frontend)
- Vuetify 3.7.5 (framework UI)

**Dependências principais:**
- @tensorflow/tfjs (machine learning e IA)
- @tiptap/* (editor de texto avançado)
- ApexCharts 3.54.1 e Chart.js 4.5.0 (visualização de dados)
- Socket.io-client 4.8.1 (comunicação em tempo real)
- Axios 1.10.0 (HTTP client)
- Vue Router 4.5.0 (roteamento)
- Vue LazyLoad 3.0.0 (lazy loading de imagens)
- @vueuse/* (composables Vue para funcionalidades comuns)
- Lodash-es 4.17.21 (utilitários)
- Marked 16.2.0 (processamento markdown)
- Floating UI (componentes UI interativos)

**Dependências de desenvolvimento:**
- ESLint, Prettier, Stylelint (padronização de código)
- TypeScript suporte via tsx
- Vite plugins para otimização e build
- Iconify (gerenciamento de ícones)
- PostCSS (processamento CSS)

**Integrações e serviços:**
- Integração direta com bancos de questões externos (REVALIDA FÁCIL, INEP, Medway)
- Firebase Authentication e Firestore
- Firebase Hosting com configuração de segurança
- Google APIs (GAPI, reCAPTCHA)
- WebSocket para comunicação em tempo real
- Backend em Node.js/Cloud Run

**Configuração e padrões:**
- Configuração centralizada em package.json, firebase.json, .env.example
- Estrutura modular por especialidade e funcionalidade
- Build múltiplos para diferentes ambientes (local, cloud, staging)
- Scripts para migração, validação e administração do banco de dados

**Frameworks de teste:**
- Testes automatizados podem ser expandidos conforme necessidade (não identificado framework específico nos arquivos atuais)
